PROJETOS RECENTES

Limpa Salvador

O Limpa Salvador é uma plataforma web que tem como objetivo facilitar a denuncia de locais com acumulo de lixo na cidade de Salvador. Na plataforma é possível fazer denuncias de locais com acumulo de lixo, visualizar as denuncias feitas por outros usuários e fazer comentários nas denuncias.

A plataforma foi desenvolvida utilizando React.js no frontend,node.js no backend e Postgresql como banco de dados.

Declinium RPG - Criação de Fichas

Site dedicado à criação de fichas no sistema Declinium RPG, um sistema desenvolvido a partir da campanha "A aurora do Caos". Esta plataforma permite aos usuários criar, gerenciar e visualizar fichas de personagens de forma intuitiva e eficiente, facilitando a preparação e o andamento das sessões de RPG.

O site foi desenvolvido utilizando React.js para a interface do usuário, Tailwind CSS para estilização e Node.js no backend.
